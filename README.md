# College Attendance App 🎓

> A streamlined, cross-platform mobile application to manage and track college attendance efficiently.

---

## 📖 Overview
Managing student attendance can be a tedious and error-prone process. This College Attendance App simplifies the workflow by providing an intuitive digital platform for logging and monitoring attendance records. Built with Flutter, it offers a seamless native experience on mobile devices and synchronizes data in real-time using Supabase, ensuring that records are always secure, up-to-date, and accessible.

## ✨ Features
* **Real-Time Tracking:** Instantly record and update student attendance.
* **Cloud Synchronization:** Securely store and retrieve attendance data using a Supabase PostgreSQL backend.
* **Cross-Platform:** Smooth performance on both Android and iOS devices from a single codebase.
* **User-Friendly Interface:** Clean, intuitive UI designed for quick navigation by educators and administrators.

## 🛠️ Built With / Tech Stack
* **Framework:** [Flutter](https://flutter.dev/) (Dart)
* **Backend & Database:** [Supabase](https://supabase.com/)
* **Development Tools:** Android Studio, VS Code
